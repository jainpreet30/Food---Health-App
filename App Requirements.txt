🧾 6. Detailed Prompts for Developers (VERY IMPORTANT)

You can literally give these to your dev team or use yourself 👇

🔹 Prompt 1: User Profile System
Build a user onboarding system where users input age, gender, height, weight, food preference, allergies, and fitness goals.

Calculate BMI and daily calorie needs using standard formulas.

Store user data securely in MongoDB and allow updating profile anytime.
🔹 Prompt 2: Diet Recommendation Engine
Create a backend module that generates personalized meal plans based on user profile.

Inputs:
- Calories required
- Food preference (veg/non-veg)
- Goal (weight loss/muscle gain)

Output:
- Daily meals with calories and macros

Use rule-based logic initially, then allow integration with AI APIs.
🔹 Prompt 3: AI Chat Assistant
Integrate an AI chatbot that answers health-related queries.

The chatbot should:
- Access user profile
- Suggest meals and exercises
- Give personalized advice

Use OpenAI API and design prompt templates for health coaching.
🔹 Prompt 4: Food Tracking System
Develop a feature where users can log meals manually or via image upload.

Store food items, calories, and nutrients.

Show total daily calorie intake vs goal.
🔹 Prompt 5: Dashboard UI
Create a responsive dashboard showing:
- Calories consumed vs target
- Water intake
- Exercise progress
- Weekly graphs

Use charts (Recharts or Chart.js).
🔹 Prompt 6: Exercise Recommendation
Design a workout recommendation engine based on user fitness level.

Provide structured plans:
- Beginner, Intermediate, Advanced

Include duration, sets, and instructions.
🔹 Prompt 7: Notification System
Implement reminders for meals, water, and workouts.

Use push notifications and email alerts.
